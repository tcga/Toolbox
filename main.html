TCGA Toolbox

TCGA Toolbox provides a scripting environment to access, process, and analyze data from TCGA in the browser. It differs from other web apps in that the main interaction happens in the JavaScript Console.

This is a work in progress. The code is hosted on GitHub.

Modules

TCGA Toolbox can be extended by loading remote scripts – either by specifying the URL or by selecting one from our trusted authors.

Tip Try RPPA, an app for real time querying and plotting of level 3 reverse phase protein data for GBM.

Documentation

Everything in the TCGA Toolbox is scoped under the global TCGA namespace. Unless stated otherwise, the first argument of a callback function is reserved for an error object.

For end users

Load scripts

TCGA.loadScript(url, callback)

TCGA.loadScript([url, url, ...], callback)

Find data

TCGA.hub.query(query, callback)

Get data

TCGA.get(url, callback)

TCGA.getJSON(url, callback)

TCGA.getXML(url, callback)

TCGA.getRange(url, startByte, endByte, callback)

Store data temporarily

TCGA.data

Store data permanently

TCGA.store.set(key, value, callback)

TCGA.store.get(key, callback)

TCGA.store.del(key, callback)

TCGA.store.exists(key, callback)

TCGA.store.keys(callback)

TCGA.store.clear(callback)

For script authors

Register tab in main menu

TCGA.registerTab({id, title, content, switchTab})

Display notifications

TCGA.toast.info(message, duration)

TCGA.toast.success(message, duration)

TCGA.toast.error(message, duration)

Get version information

TCGA.version
